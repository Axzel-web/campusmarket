# Campus Market App

A web application that serves as a marketplace for students to buy and sell used textbooks, electronics, and other campus-related items.

## Features
- User Accounts: Users can create an account and manage their listings.
- Product Listings: Students can list used books, electronics, and other items for sale.
- Search & Filters: Search items by category.
- Messaging System: Users can contact sellers directly via in-app messaging.
- Ratings & Reviews: Buyers can leave reviews and ratings on items they've purchased.
